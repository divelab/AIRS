# Datasets (`pyg_dataset.py`)

`TDDFTv2_pyg` reads trajectory `.pt` files from `raw/`, preprocesses them into PyG `Data` objects under `processed/` (once), and then serves autoregressive training/eval samples.

The raw trajectory files are typically generated by the preprocessing pipeline in `datagen_abacus/preprocess`.

## Conventions

- `get(idx)` returns one molecule sample at one target time `t`, after applying `ar_transform(...)`.
- Coefficients are normalized:
  - `coef_0` is scaled by `rms_0`
  - `delta_coef_cond` / `delta_coef_target` are scaled by `rms_t`
- `collate_fn(...)` returns a dict with two graph granularities:
  - `molecule_data`: molecule-level tensors (one entry per molecule in batch)
  - `state_data`: occupied-state-expanded tensors (one entry per occupied state)
- Legacy note: older code may refer to `global_data` / `graph_data`; the current convention is `molecule_data` / `state_data`.
- Key index maps from `collate_fn`:
  - `molecule_data.batch`: atom -> molecule id
  - `state_data.batch`: atom-in-state -> state id
  - `state_data.state_ind_batch`: state id -> molecule id
  - `state_data.mol_batch`: atom-in-state -> molecule id
  - `state_data.state_atom_batch`: atom-in-state -> atom index in `molecule_data`

## Example: initialize dataset

```python
from orbevo.datasets.pyg_dataset import TDDFTv2_pyg

dataset = TDDFTv2_pyg(
    root="data/QM9_tddft",
    time_start=0,
    time_cond=8,
    time_future=8,
    time_gap=1,
    T=100,
    rms_0=0.0123,  # use your config value
    rms_t=0.0456,  # use your config value
)
```

## Example: use `collate_fn`

```python
from torch.utils.data import DataLoader
from orbevo.datasets.pyg_dataset import collate_fn

loader = DataLoader(
    dataset,
    batch_size=16,
    shuffle=True,
    num_workers=4,
    collate_fn=collate_fn,
)

batch = next(iter(loader))
molecule_data = batch["molecule_data"]
state_data = batch["state_data"]
```
